# Sunucu Kuralları

> Hard RP · Text-based · Türkçe  
> Oyun içi özet: `/kurallar`

---

## 1. Genel

1. Bu sunucu **yazı tabanlı roleplay** sunucusudur. İletişim chat komutları ile yapılır; **sesli konuşma (voice) kapalıdır**.
2. Her oyuncunun **1 karakteri** vardır. İkinci karakter veya karakter silme yoktur (CK admin prosedürü).
3. Gerçekçi isim zorunludur. Meme isim, ünlü isim, rakam/symbol içeren isimler yasaktır.
4. Minimum karakter yaşı: **18**.

## 2. Roleplay kuralları

| Kural | Açıklama |
|-------|----------|
| **IC / OOC** | Karakter içi konuşma IC'dir. OOC için `/b` kullanın. |
| **Powergaming** | Gerçekte yapamayacağınız eylemi yazmayın veya zorlamayın. |
| **Metagaming** | OOC'den (Discord, stream vb.) öğrendiğiniz bilgiyi IC kullanmayın. |
| **Fear RP** | Silah/tehdit altındayken mantıklı davranın; kahramanlık yapmayın. |
| **NLRP** | Ölüm sonrası olayı unutursunuz; intikam RP'si yasaktır (CK hariç). |
| **RDM / VDM** | Sebepsiz öldürme ve araçla ezme yasaktır. |
| **Combat logging** | RP devam ederken disconnect atmayın. |

## 3. Chat kullanımı

| Komut | Kullanım |
|-------|----------|
| `T` + mesaj | Normal konuşma (~20m) |
| `/me` | Eylem |
| `/do` | Ortam |
| `/s` | Fısıltı (~3m) |
| `/w` | Bağırma (~40m) |
| `/b` | Yerel OOC |
| `/911` | Acil çağrı (PD / EMS) |

Global chat yoktur. Her eylemi `/me` ile destekleyin.

## 4. Acil servisler

- `/911 [mesaj]` — görevdeki polis ve sağlık personeline text çağrı gider.
- Troll 911 yasaktır; cooldown uygulanır.
- PD/EMS yalnızca **görevdeyken** (on duty) çağrı alır.

## 5. Ekonomi & sistem

- Başlangıç parası sınırlıdır; bedava gelir yoktur.
- Soygun, uyuşturucu ve arcade scriptler kapalıdır — RP ile yönetilir.
- Araç kontağı, kilit ve km takibi roleplay'e tabidir.

## 6. Ceza sistemi (özet)

| İhlal | İlk | Tekrar |
|-------|-----|--------|
| Hafif OOC ihlal | Uyarı | Kick |
| Metagaming / PG | Kick | Ban (1–7 gün) |
| RDM / VDM | Ban (1–7 gün) | Kalıcı |
| Hile / exploit | Kalıcı ban | — |

Detay admin takdirindedir.
